Frequently Asked Questions

What is a bounce checker?

In TF2 rocket jumping, certain physics bugs allow players to touch teleport triggers without being warped away. While players often trigger these by accident, known as "random bounces", it is possible to execute them consistently using Low Input Strategies (LIS). A bounce checker is a tool for finding consistent, executable LIS "setups."

How does this bounce checker work?

The tool works by pairing movement binds (like +forward) with action binds (like +attack). To perform a setup, you hold down the movement bind first, followed by the action bind. For simpler setups, a single tap of the action bind is enough. For more complex setups, such as when holding the action button triggers a continuous pogo, you may need to manually crouch or uncrouch between firing each rocket.

You can look for specific setups using the "Preferences" tab. If you are a beginner, it is highly recommend maxing out the "Simple" preference. If you are an advanced jumper, try increasing the weight of "Fully automated" setups. These are fancy setups where you can simply hold down M1 to hit the bounce, which is a major feature of this tool.

What to do if a setup isn't working?

A setup might not work for many reasons. Some things in particular to consider is:

  • Make sure you are looking straight down. Do not move your mouse.
  • Make sure /speedo is reporting the same speeds shown at the bottom of the setup. It should match perfectly.
  • Make sure you are crouched/uncrouched at the right times. The instructions of when to be crouched/uncrouched is given by the colors of the speed tags shown at the bottom. Using /skeys can help with figuring out if you are crouched or not.
  • For diagonal setups, it is highly recommended to use a movement bind. Without a bind your speed might be oscillating, causing the setup to fail.
  • Make sure to hold down the movement bind for the entire duration of the setup. This is mostly relevant for setups where you touch the ground inbetween rockets.
  • Make sure you release the action bind after pressing it. Unlike the movement bind, you are not supposed to hold the action bind down. If the setup requires you to shoot additional rockets, then you should shoot them by holding down m1.

If you are still unnable to perform the setup, then send me (SUthrey) a DM on discord with the height and the ID of the setup.

How was this tool made?

This is a long-standing passion project of mine. By analyzing TF2 source code (open-sourced by Valve themselves), I've built a custom, physics-accurate simulation of TF2's movement engine in Python. I then created this bounce checker on top of that simulation, programmatically testing hundred of thousands of LIS combinations to see which ones produce a bounce. Every setup on this website was generated by that Python tool. While the website currently displays pre-calculated results rather than running the engine live, I hope to port the simulation to JavaScript in the future to allow on-demand generation.
